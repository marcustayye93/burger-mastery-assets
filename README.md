# Burger Mastery Assets Repository

This repository stores all approved Burger Mastery photography and visual assets.

## Naming convention

Use:

`BM-<CATEGORY>-<NUMBER>-<descriptive-slug>.webp`

Examples:

- `BM-MEAT-001-beef-hero.webp`
- `BM-PATTY-001-thumb-dimple.webp`
- `BM-COOK-001-perfect-crust.webp`

## Rules

- Do not use vague filenames such as `image1`, `final`, or `burger-photo`.
- Do not rename existing assets unless explicitly instructed.
- Do not reorganize folders without approval.
- Do not embed text, logos, UI, or watermarks into images.
- Preserve the approved Burger Mastery photography style.
